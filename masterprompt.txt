Never replaces the driver

Physics envelopes are hard ceilings

Learned values may only reduce conservatism, never exceed physics

No real-time machine learning in safety-critical loops

Voice output must be deterministic and finite

Mid-corner speech is forbidden unless instability is imminent

All learning must be local, slow, reversible, and confidence-gated

UI must never block telemetry, physics, or voice arbitration

If any invariant is violated → redesign the system instead of proceeding.
